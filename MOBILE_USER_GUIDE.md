# 手机端使用指南 - Zhineng Bridge 智桥

## 📱 快速开始

### 前提条件

1. **电脑端**：运行 Zhineng Bridge Relay Server
2. **手机端**：可以访问电脑的 IP 地址
3. **网络**：手机和电脑在同一局域网

---

## 步骤 1: 查看电脑 IP 地址

### Windows
```cmd
ipconfig
```
找到 "无线局域网适配器" 下的 "IPv4 地址"，例如：`192.168.31.99`

### macOS
```bash
ifconfig | grep "inet "
```
找到 `en0` 下的 IP 地址

### Linux
```bash
ip addr show | grep "inet "
```
找到 `wlp` 或 `wlan` 开头的 IP 地址

---

## 步骤 2: 确认服务器已启动

在电脑上运行：
```bash
cd /home/ai/zhineng-bridge/relay-server
node server.js
```

看到以下输出表示成功：
```
Zhineng Bridge Relay Server running on port 8001
WebSocket endpoint: ws://localhost:8001
HTTP server: http://localhost:8001
```

---

## 步骤 3: 在手机浏览器中访问

### 方法 1: 直接输入地址
在手机浏览器地址栏输入：
```
http://192.168.31.99:8001/mobile-client.html
```
（替换为你的电脑 IP 地址）

### 方法 2: 扫描二维码（如果有）
- 在电脑上生成二维码包含 URL
- 用手机扫码直接访问

### 方法 3: 添加到主屏幕
1. 打开浏览器菜单
2. 选择"添加到主屏幕"
3. 下次可以直接从桌面打开

## 常见问题

### Q1: 无法连接到服务器？

**检查清单**：
- [ ] 确认电脑服务器正在运行
- [ ] 确认手机和电脑在同一网络
- [ ] 确认 IP 地址正确（不是 localhost）
- [ ] 确认使用正确的协议（ws://）
- [ ] 尝试 ping 电脑 IP：`ping 192.168.31.99`
- [ ] 检查防火墙设置

**解决方法**：
```bash
# 在电脑上开放端口
sudo ufw allow 8001

# 或者临时关闭防火墙测试
sudo ufw disable
```

---

### Q2: 连接后收不到消息？

**可能原因**：
1. 频道名称不匹配
2. AI 工具未发送消息
3. 网络问题

**解决方法**：
1. 检查频道名称是否为 `ai-assistant`
2. 确认 AI 工具已启动并发送消息
3. 查看浏览器控制台是否有错误

---

### Q3: 按钮点击没反应？

**解决方法**：
1. 刷新页面
2. 重新连接
3. 检查浏览器是否支持（推荐 Chrome、Safari）
4. 查看网络连接状态

---

### Q4: 输入框无法输入？

**可能原因**：
- 未连接服务器

**解决方法**：
1. 确认状态为"已连接"
2. 如果未连接，点击"连接"按钮

---

### Q5: 如何修改服务器地址？

**步骤**：
1. 点击右上角的"断开"按钮
2. 点击"连接"按钮
3. 修改服务器地址
4. 点击"保存并连接"

---

### Q6: 手机和电脑不在同一网络怎么办？

**解决方案**：

**方案 1: 使用内网穿透工具**
- 推荐工具：ngrok、frp、localtunnel
- 示例：
  ```bash
  ngrok http 8001
  ```
  得到公网地址：`https://xxxx.ngrok.io`
  手机访问：`https://xxxx.ngrok.io/mobile-client.html`

**方案 2: 使用云服务器**
- 部署 Relay Server 到云服务器
- 手机直接访问云服务器地址

## 安全提示

### 1. 使用 HTTPS（生产环境）

在生产环境中，建议使用 `wss://` 协议：

```javascript
// 修改移动端连接地址
const serverUrl = 'wss://yourserver.com:8443';
```

### 2. 设置密码（可选）

为 Relay Server 添加认证：

```javascript
// 在 server.js 中添加认证
wss.on('connection', (ws, req) => {
  const token = req.headers['authorization'];
  if (!isValidToken(token)) {
    ws.close(1008, 'Invalid token');
    return;
  }
  // 继续处理
});
```

### 3. 局域网使用

当前配置仅适用于局域网使用，不对外开放。

## 总结

### 使用流程回顾

```
1. 电脑启动服务器
   ↓
2. 手机访问 http://<电脑IP>:8001/mobile-client.html
   ↓
3. 点击"连接"按钮，输入服务器地址
   ↓
4. 等待 AI 工具发送问询
   ↓
5. 点击选项按钮或输入自定义回答
   ↓
6. AI 工具接收响应并继续执行
```

### 关键要点

✅ **确保 IP 正确**：不要使用 localhost，使用电脑的实际 IP
✅ **确保在同一网络**：手机和电脑在同一 WiFi
✅ **确保服务器运行**：电脑上的 server.js 必须在运行
✅ **使用快捷指令**：快速控制 AI 工具
✅ **及时响应问询**：提高开发效率
