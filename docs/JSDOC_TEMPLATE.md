# JSDoc 注释模板

## 函数注释模板

```javascript
/**
 * 函数简短描述
 * @param {type} paramName - 参数描述
 * @returns {type} 返回值描述
 * @throws {Error} 可能抛出的错误
 * @example
 * const result = functionName(param);
 */
function functionName(paramName) {
    // 实现
}
```

## 类注释模板

```javascript
/**
 * 类简短描述
 * @class
 * @classdesc 详细的类描述
 */
class ClassName {
    /**
     * 构造函数
     * @param {type} param - 参数描述
     */
    constructor(param) {
        // 实现
    }
}
```

## 事件处理器注释模板

```javascript
/**
 * 事件处理器描述
 * @param {Event} event - 事件对象
 * @returns {void}
 */
function handleEvent(event) {
    // 实现
}
```
