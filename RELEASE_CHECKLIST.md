# 智桥（Zhineng-bridge）发布检查清单

本文档提供了发布智桥新版本的完整检查清单，确保发布质量。

## 发布前准备

### 1. 代码质量检查

- [ ] **代码审查**
  - [ ] 所有 PR 已通过审查
  - [ ] 代码风格一致（遵循项目规范）
  - [ ] 无明显的代码异味或反模式
  - [ ] 所有 TODO 和 FIXME 已处理或记录

- [ ] **测试覆盖**
  - [ ] 单元测试全部通过（`python3 -m pytest tests/unit/ -v`）
  - [ ] 集成测试全部通过（`python3 -m pytest tests/integration/ -v`）
  - [ ] E2E 测试全部通过（`python3 -m pytest tests/e2e/ -v`）
  - [ ] 测试覆盖率 >= 70%（当前目标：82%）
  - [ ] 无跳过或失败的测试

- [ ] **静态分析**
  - [ ] 无 lint 错误
  - [ ] 无类型检查错误（如果使用 mypy）
  - [ ] 无安全漏洞警告

### 2. 文档完整性检查

- [ ] **核心文档**
  - [ ] README.md 已更新（版本号、功能列表）
  - [ ] CHANGELOG.md 已更新（详细记录变更）
  - [ ] VERSION 文件已更新
  - [ ] QUICKSTART.md 内容完整且准确
  - [ ] USAGE_GUIDE.md 内容完整且准确
  - [ ] TROUBLESHOOTING.md 内容完整且准确

- [ ] **API 文档**
  - [ ] docs/API.md 已更新
  - [ ] 所有公开 API 都有文档说明
  - [ ] 示例代码可运行

- [ ] **示例文件**
  - [ ] examples/basic_usage.py 可运行
  - [ ] examples/cursor_config.md 内容准确
  - [ ] examples/claude_config.md 内容准确
  - [ ] examples/README.md 内容完整

### 3. 功能验证

- [ ] **核心功能**
  - [ ] relay-server 启动正常
  - [ ] session_manager 启动正常
  - [ ] WebSocket 连接正常
  - [ ] 会话创建/停止/删除正常
  - [ ] 多会话管理正常
  - [ ] 心跳机制正常

- [ ] **Web UI**
  - [ ] 主页面加载正常
  - [ ] 会话列表显示正常
  - [ ] 创建会话功能正常
  - [ ] 删除会话功能正常
  - [ ] 实时输出显示正常
  - [ ] 响应式设计正常（桌面、平板、移动端）

- [ ] **AI 工具支持**
  - [ ] Crush 工具集成正常
  - [ ] Claude Code 工具集成正常
  - [ ] Cursor 工具集成正常
  - [ ] 其他工具集成正常（iFlow CLI, Trae, Droid, OpenClaw, GitHub Copilot）

- [ ] **性能**
  - [ ] Session 创建时间 < 100ms
  - [ ] WebSocket 连接时间 < 50ms
  - [ ] 页面加载时间 < 2s
  - [ ] 内存使用 < 100MB
  - [ ] 无明显性能瓶颈

### 4. 安全检查

- [ ] **依赖安全**
  - [ ] 所有依赖已更新到最新稳定版本
  - [ ] 无已知的安全漏洞（使用 `pip check` 或 `safety check`）
  - [ ] 依赖版本已固定（如果使用 requirements.txt）

- [ ] **代码安全**
  - [ ] 无硬编码的密钥或密码
  - [ ] 无 SQL 注入风险
  - [ ] 无 XSS 风险
  - [ ] 敏感数据已加密

- [ ] **权限**
  - [ ] 文件权限设置正确
  - [ ] 网络端口权限正确
  - [ ] 敏感操作有适当的权限检查
